## API Reference

### Props

| Prop       | Type     | Description                    |
| ---------- | -------- | ------------------------------ |
| strings    | string[] | An array of strings to type    |
| speed      | object   | Typing speed configuration     |
| loop       | boolean  | Whether to loop the animation  |
| autoStart  | boolean  | Start typing automatically     |
| pauseFor   | number   | Pause duration in milliseconds |
| direction  | string   | Text direction (LTR or RTL)    |
| textEffect | string   | Text effect to apply           |
| cursor     | object   | Cursor configuration           |
